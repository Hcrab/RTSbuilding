# 开始使用

## 打开 RTSBuilding

游戏内按 `G`（默认键）打开 RTSBuilding，进入俯视建造视角。

再次按 `G` 或 `Esc` 关闭 RTS 界面。

## 界面布局

RTSBuilding 的界面主要分为三块：

- 顶部栏：切换交互、储存绑定、漏斗、旋转等模式，打开快速建造、范围隐藏、设置和教程。
- 底部栏：查看储存物品、选择方块和工具、使用蓝图、远程合成，以及打开插件管理。
- 侧边悬浮窗：打开箱子或机器界面时出现，用于将容器物品存入 RTS 储存。

## 首次使用指南

如果服务器没有启用生存平衡，可以先绑定一个箱子作为 RTS 储存：

1. 打开 RTSBuilding。
2. 点击顶部栏的储存绑定按钮。
3. 准星对准场景中的箱子。
4. 左键点击箱子。
5. 确认顶部提示显示“已绑定储存”。
6. 确认底部栏已显示箱内物品。

绑定后，左键点击底部栏的方块，把它设为当前放置物品。准星对准放置位置并右键点击，就可以从远处放置。

## 下一步

- 想铺路、拉墙或搭盒子，看“顶部栏 / 快速建造”。
- 想挖一大片区域或连锁清树，看快速建造里的“范围破坏”。
- 想隐藏屋顶、墙体或楼板，方便从俯视角继续建造，看“顶部栏 / 范围隐藏”。
- 想管理绑定的箱子、机器和仅提取目标，看“顶部栏 / 模式和按钮 / 储存绑定”。
- 想固定常用材料和工具，看“底部栏 / 储存空间”。

## 生存平衡

如果界面缺失按钮，通常说明服务器启用了生存平衡。新版生存平衡使用 RTS 插件决定可用功能。

打开背包，点击屏幕顶部的 `RTS` 按钮，或在 RTS 底部栏点击“插件管理”，把背包里的插件安装进去。已安装插件决定你能使用哪些 RTS 功能。

只安装“RTS 控制核心”时，你可以进入 RTS 视角、进行基础交互，并获得 16 格基础操作半径；远程放置、远程挖掘、储存整合、蓝图、范围破坏和范围隐藏需要对应插件。

生存平衡下还需要设置 RTS 家园。打开背包，点击屏幕顶部的“RTS 家园”按钮设置家园位置；RTSBuilding 会以家园为中心决定可使用 RTS 功能的范围。

详细机制请看“生存模式机制”里的“RTS 插件与生存平衡”和“RTS 家园与范围限制”。

## 教程与提示

点击界面各区域的 `i` 按钮即可查看对应教程。

顶部提示会实时显示当前模式和下一步操作要求，例如已选物品、绑定储存状态、快速建造等待 A/B 点，或范围破坏等待确认。
