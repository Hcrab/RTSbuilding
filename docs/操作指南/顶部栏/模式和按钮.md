# 模式和按钮

顶部栏里的按钮会改变 RTSBuilding 的操作方式。最重要的是左侧的模式按钮：你点了哪个模式，接下来点击世界里的方块就会触发对应的动作。

## 交互

交互是默认模式，也最接近原版游玩。

在交互模式下：

- 右键优先使用方块、机器、箱子或门。
- 如果没有可用互动，右键会尝试放置当前选中的物品。
- 按住 `Shift` 再右键时，会优先尝试放置当前选中的物品，再回退到互动。
- 左键可以挖掘方块。

你也可以对准世界中的方块按下“选取方块”键，默认是鼠标中键，把它直接选为接下来要放置的物品。

## 储存绑定

储存绑定用于告诉 RTSBuilding：从哪些容器或储存系统里拿取材料，以及把物品存回哪里。

使用方法：

1. 点击顶部栏的储存绑定按钮。
2. 对准箱子、桶、抽屉、仓库控制器或其他储存方块。
3. 左键点击它，把它绑定为可存入、可提取的 RTS 储存。
4. 绑定成功后，顶部提示文字会显示已绑定的储存名称。

右键也可以绑定储存，但右键绑定是“仅提取”。这种绑定适合只希望 RTSBuilding 从那里拿东西，不希望它把物品存回去的容器。

鼠标悬停在储存绑定按钮上，会出现“查看绑定详情”。点击后会打开“储存绑定详情”窗口。

这个窗口可以查看所有已绑定方块，并直接管理它们：

- 方块名称和坐标：确认每个绑定对应的位置。
- 优先级：调整多个绑定储存的尝试顺序。普通使用保持 `0` 即可；需要精细分流时，存入会优先尝试较高数值，取出会优先尝试较低数值。
- 提取：显示“是”时表示仅提取，RTSBuilding 不会把物品存回这个目标；显示“否”时表示可存可取。
- 解绑：移除这一个储存绑定。

绑定很多储存时，详情窗口可以滚动查看全部条目。

进入储存绑定模式后，普通的右键互动会被暂时禁用。

## 漏斗

漏斗模式用于在 RTS 视角下收集地面掉落物。把鼠标移动到掉落物附近，漏斗会把周围掉落物吸入已绑定的 RTS 储存。

使用方法：

1. 先绑定一个可以存入物品的储存。
2. 点击顶部栏的漏斗按钮，或按住快速漏斗热键。
3. 把 RTS 鼠标移动到地面掉落物附近。
4. 掉落物会被吸入 RTS 储存空间。

默认快速漏斗热键是 `F`。按住 `F` 时临时进入漏斗模式，松开后回到之前的模式。

## 旋转

旋转模式用于从 RTS 俯视镜头下改变方块朝向。

使用方法：

1. 点击顶部栏的旋转按钮。
2. 对准需要旋转的方块。
3. 右键点击方块。

适用于调整楼梯、机器等可旋转方块的朝向。

## 快速建造

快速建造会打开批量操作面板，里面包含“范围建造”和“范围破坏”。范围建造用于一次放置一组方块；范围破坏用于按形状或连锁规则批量挖掉方块。

详细步骤请看“顶部栏 / 快速建造”。

## 范围隐藏

范围隐藏会打开一个小窗口，用来创建、选择和调整隐藏方块区域。

它常用于把屋顶、墙体或楼板临时隐藏，让 RTS 的黄色选择框和右键射线直接穿过去，命中内部方块。

详细步骤请看“顶部栏 / 范围隐藏”。

## 区块显示

区块显示按钮会显示或隐藏区块边界。

这个功能适合规划农场、机器阵列、基地模块和需要按区块边缘对齐的建筑。

再次点击按钮可以关闭区块显示。

## 教程按钮

顶部栏里的 `i` 按钮会打开当前区域的教程说明。

不熟悉某个按钮或面板时，点击 `i` 查看教程。教程弹窗会覆盖在 RTS 界面上方，关闭后可以继续操作。

## 设置按钮

右侧齿轮按钮会打开 RTS 控制设置。

这里可以调整 RTS UI 缩放、镜头手感、自动入库、Shift 存入、工具保护、预览动画等选项。

这里的设置只影响你个人的操作习惯和画面效果。建造是否消耗物品、服务器规则等生存平衡限制，不受这些设置影响。
