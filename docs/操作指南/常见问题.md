# 常见问题

## Q：如何设置储存绑定的存取权限？

左键绑定是“存取”，右键绑定是“仅提取”。

绑定后要修改，鼠标悬停在顶部栏“储存绑定”按钮上，点击“查看绑定详情”。“提取 = 是”表示仅提取；“提取 = 否”表示可存可取。更完整的说明见“顶部栏 / 模式和按钮 / 储存绑定”。

## Q：为什么某些按钮不见了？

通常是服务器启用了生存平衡，但你还没有安装对应 RTS 插件。

打开背包，点击顶部的 `RTS` 按钮，或在 RTS 底部栏点击“插件管理”。安装“RTS 控制核心”只能获得基础 RTS 视角和基础交互；远程建造、储存、蓝图、范围破坏、范围隐藏等功能需要对应插件。

## Q：快速建造里为什么找不到原来的连锁挖掘按钮？

连锁挖掘已经合并到快速建造面板里。

打开顶部栏的“快速建造”，切到“范围破坏”，再选择“连锁”。之后左键点击目标方块，就会按当前上限破坏相连的同类方块。

范围建造、范围破坏和连锁破坏现在都在同一个面板里。
## Q：“连接”按钮是做什么的？

当形状是“直线”或“墙”时，“连接”会让斜向路线补齐拐角，保证线段连续。

它适合铺路、拉墙、放管线、放缆线，也适合在范围破坏里挖出连续通道。
