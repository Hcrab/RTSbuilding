# 范围隐藏

范围隐藏用于临时隐藏一块方形区域里的已有方块，并让 RTS 的射线穿过这块区域。

它适合处理屋顶、楼板、墙体、树冠等挡住俯视建造视线的东西。范围隐藏不会真的删除方块，只影响当前客户端在 RTS 视角里的显示和点击命中。

生存平衡开启时，需要先安装游戏内物品“范围隐藏插件”。创造模式可以直接使用。

## 打开范围隐藏

在 RTSBuilding 顶部栏点击“范围隐藏”按钮。

打开后会出现一个小窗口，右上角红色叉号可以关闭范围隐藏模式。关闭时，如果你已经完成一个完整的框选草稿，会自动创建这个隐藏盒；半截草稿会被取消。

## 创建隐藏盒

创建方式有两种。

方式一：框选区域

1. 左键点击第一个角点。
2. 左键点击平面上的对角点。
3. 用滚轮调整高度。
4. 按 `Enter` 创建隐藏盒。

方式二：单点创建

如果只想创建 1x1x1 的隐藏盒，左键点击一个方块后直接按 `Enter`。

创建后，隐藏盒会让区域内原本存在的方块消失。你的黄色选择框、放置射线和交互射线会跳过这些方块，继续命中后面的真实方块。

## 调整隐藏盒

左键点击已有隐藏盒可以选中它。选中后，盒子的六个方向会出现箭头。

调整方式：

1. 左键点击一个箭头。
2. 箭头变成金色后，滚轮只用于调整这个方向的面，不再缩放镜头。
3. 向上滚扩大这个方向，向下滚收回这个方向。
4. 按 `Esc` 退出当前箭头调整。

六个箭头分别对应 +X、-X、+Y、-Y、+Z、-Z。箭头可以隔着被遮挡的方块选中，所以即使隐藏盒的一面埋在墙里，也可以继续调整。

## 删除隐藏盒

选中隐藏盒后，点击范围隐藏窗口里的“删除”，或按 `Delete` / `Backspace`。

## 与建造交互的关系

范围隐藏只影响“创建隐藏盒时已经在那里的方块”。在隐藏区域内新放置的方块会正常显示，方便你确认刚放下的箱子、机器或装饰方块。

范围隐藏也会影响交互命中：右键互动、放置和黄色选择框会跳过被隐藏的旧方块。这样你可以先隐藏屋顶，再直接操作屋内的方块。

如果不再需要隐藏效果，关闭范围隐藏模式或删除对应隐藏盒即可。它不是服务器破坏操作，也不会替你节省挖掘步骤。
